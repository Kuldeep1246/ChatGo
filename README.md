🚀 MERN Chat App


🎥 Watch Full Tutorial on YouTube

A real-time chat application built using the MERN stack, enhanced with Socket.io for instant messaging, and styled with TailwindCSS + DaisyUI. This app features full authentication, online status tracking, image uploads, and a smooth developer experience.

✨ Highlights
⚙️ Tech Stack: MongoDB, Express, React, Node.js, Socket.io

🔐 Authentication & Authorization with JWT

📡 Real-time messaging using Socket.io

🟢 Online user presence

🌍 Global state management with Zustand

🧠 Robust error handling on both client and server

🎨 UI Components with TailwindCSS & DaisyUI

🚀 Free Deployment walkthrough included

🔧 Setup Instructions
1. Clone the Repository
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
2. Set Up .env File
Create a .env file in your root directory and add the following:

env
Copy
Edit
MONGODB_URI=your_mongodb_connection_string
PORT=5001
JWT_SECRET=your_jwt_secret

CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret

NODE_ENV=development
💡 Make sure to secure your .env file and never commit it to version control.

⚙️ Build & Run
Build the App
npm run build
Start the Server
npm start
